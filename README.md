# Groupomania

OpenClassrooms - Développeur Web - Projet 7
Créez un réseau social d'entreprise

## Lancement

### Backend

- Completer le fichier .env
- Crée une base de donnée du nom donnée dans le .env DB_NAME

```bash
cd Backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run serve
```
